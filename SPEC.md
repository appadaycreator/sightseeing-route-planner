# 観光ルート最適化ツール【無料】自動作成ツール - 技術仕様書

## 概要

**サービス名**: Sightseeing Route Planner
**バージョン**: 1.3.0（2026-06-14改善版）
**URL**: https://appadaycreator.com/sightseeing-route-planner/

訪問スポットを入力して効率的な観光順序と移動時間・費用を計算。登録不要・完全無料でご利用いただけます。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元（スポット・交通手段・開始時刻）
- **クリア**: 「データクリア」ボタンで削除可能

## 主な機能（v1.1.0）

### コア機能
- **スポット入力**: 最大10個のスポットを追加（名前・滞在時間）
- **交通手段選択**: 徒歩・自転車・車・公共交通から選択
- **ルート計算**: スポット間距離を推定し、移動時間・費用を自動計算
- **開始時刻**: オプションで開始時刻を指定すると終了時刻も自動計算

### 改善点（v1.1.0）
1. **計算結果の再現性**: 毎回同じ入力で同じ結果（ランダム性を排除）
2. **データ永続化**: localStorage対応で前回の入力を自動復元
3. **交通費計算**: 公共交通の現実的な計算（基本運賃¥200 + ¥100/km）
4. **スポット並び替え**: ↑↓ボタンで順序を変更可能
5. **バリデーション強化**: 複数のエラーチェック（滞在時間0～240分範囲等）

### 改善点（v1.2.0 - 2026-06-10）
1. **入力フォーム UX改善**
   - 各入力項目に `aria-label` を追加（アクセシビリティ向上）
   - スポット名・滞在時間の入力欄に具体例を placeholder に表示（「京都駅・伏見稲荷大社」「30」など）
   - 開始時刻入力に説明テキスト「指定すると終了時刻も計算します」を追加
   - 滞在時間の制約を 0～240分に制限（max 属性）

2. **バリデーション強化**
   - スポット名未入力時に「スポット○の名前を入力してください」と具体的なエラー表示
   - 滞在時間が範囲外の場合に「スポット○の滞在時間は0～240分の範囲で入力してください」と指示

3. **localStorage を使用したデータ保存・復元**
   - スポット一覧・交通手段・開始時刻を自動保存
   - ページリロード時に前回の入力値を自動復元
   - 「データクリア」ボタンで全データを一括削除
   - 復元時に「前回のデータを復元しました」メッセージを表示（成功メッセージ）

### 改善点（v1.3.0 - 2026-06-14）
1. **M1: 診断/計算後の文脈CTAボタン設置**
   - 計算結果表示下にコンテキストCTA配置（「このルートで国内旅行プランを検索」「滞在予定日数でホテル・宿泊施設を探す」）
   - 推定滞在日数を自動計算して表示（「推定滞在日数: 2日間の旅行プランを検索できます」）
   - A8.net アフィリエイトリンク保持（宿泊施設・旅行プラン検索へのリンク）

2. **M2: コア機能のUX改善（入力UI・バリデーション）**
   - 入力フォーム外枠に視覚的ハイライト（デフォルト: グレー枠、エラー: 赤枠）
   - スポット入力時に入力状態をアイコン表示（✓ で入力完了、※ で未入力を視覚化）
   - リアルタイム予算バリデーション: 入力時に即座にエラーメッセージ表示
   - エラーメッセージに より具体的な説明を追加（例: 「予算は0より大きい数値を入力してください」）

3. **M6: 目標設定・達成度トラッキング**
   - 目標予算のlocalStorage保存・復元（前回設定した予算目標を自動復元）
   - 達成率の視覚化: プログレスバー表示（目標金額に対する進捗を色付き棒グラフで表示）
   - 前回との比較表示: 「前回より○%改善/悪化」と自動計算・表示
   - 複数回の計算で目標達成度の推移を追跡可能

### 距離推定アルゴリズム
- 1スポット: 0km
- 2スポット: 3km
- 3～5スポット: 5km × (スポット数 - 1)
- 6スポット以上: 7km × (スポット数 - 1)

### 交通手段別費用計算
| 手段 | 計算方式 |
|-----|--------|
| 徒歩 | ¥0 |
| 自転車 | ¥0 |
| 車 | ¥15/km × 距離 |
| 公共交通 | ¥200（基本） + ¥100/km × (距離-2km) |

## 使い方

1. 入力フォームの各項目を確認する
2. 必要な数値・条件をすべて入力する
3. 計算ボタンをクリックして結果を取得する
4. 計算結果と詳細な内訳を確認する
5. 条件を変えて複数パターンを比較する

## よくある質問（FAQ）

**Q: 観光ルート最適化ツールは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 計算結果は正確ですか？**

参考値としてご活用ください。重要な判断の際は専門家・公式機関にご確認ください。

**Q: 何回でも計算できますか？**

はい、回数制限なく何度でもご利用いただけます。条件を変えて比較にお使いください。

**Q: 入力したデータはどこかに保存されますか？**

いいえ。すべての計算はブラウザ内で処理され、データのサーバー送信は行いません。

**Q: スマートフォンで利用できますか？**

はい、PC・スマートフォン・タブレットすべてに対応しています。
